# Java Smart Lab Automation System

A console-based Smart Lab Automation System built using Java.

## Features

- Control electric appliances
- Remote control integration
- Appliance status management
- Power consumption tracking
- Lab device automation

## 🛠 Technologies Used

- Java
- OOP
- Abstract Classes
- Interfaces
- Inheritance
- Polymorphism

## Concepts Demonstrated

### Abstract Class
- ElectricAppliance

### Inheritance
- Fan
- TubeLight
- AC
- Projector

### Interface
- RemoteControllable

### Composition
- RemoteControl → ElectricAppliance

### Method Overriding
- turnOn()
- turnOff()
- performPrimaryFunction()
